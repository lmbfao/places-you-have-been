# Places You've Been

**Author:** Christine Karuga

## Description

"Places You've Been" is a simple web application that allows users to track and display destinations they have visited. Each destination includes details such as:
- location
- landmark(s)
- optional image(s)
- Time of visit
- Notes about the place

The app is built using:
- HTML
- CSS
- JavaScript

The UI displays a clickable list of destinations, and selecting a place updates the details section.

The colour scheme is soft pink and green.

## Project Setup Instructions

1. Clone the repository to your local machine:

  ```bash
  git clone https://github.com/lmbfao/places-you-have-been.git
  ```

2. Navigate to the project folder:

  ```bash
  cd places-youve-been
  ```

3. Open "[index.html](index.html)" in your browser.

4. Optional: Add images to the "[images](images)" folder and update the place objects in "[script.js](script.js)"to display them.
